Display

Berikut adalah contoh penggunaan properti display pada CSS:

  1. Display: none

    Elemen dengan display:none tidak akan muncul di halaman.

    Elemen ini tidak akan ditampilkan di halaman.

  2. Display: inline

    Elemen inline berada di baris yang sama dengan elemen lain, seperti teks.

    Elemen inline Elemen inline lain

  3. Display: block

    Elemen block memenuhi lebar penuh kontainer dan dimulai di baris baru.

    Elemen block
    Elemen block lain

  4. Display: inline-block

    Elemen inline-block berada di baris yang sama seperti inline, tapi bisa diberi ukuran.

    Elemen inline-block
    Elemen inline-block lain

  5. Display: flex

    Elemen flex digunakan untuk membuat tata letak fleksibel yang bisa disesuaikan.

    Elemen 1
    Elemen 2
    Elemen 3

  6. Display: grid

    Elemen grid digunakan untuk membuat tata letak grid yang terdiri dari baris dan kolom.
