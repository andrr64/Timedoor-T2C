Pertemuan 10 - CSS Selector

CSS selector adalah aturan dalam CSS untuk memberikan gaya pada elemen, kelas, atau id tertentu.

Jenis-Jenis Selector

1. Selector Universal

Selector universal digunakan untuk memberikan gaya pada semua elemen

* {
 color: blue;
}

Arti dari kode di atas adalah semua elemen (*) akan diubah warnanya menjadi biru (color: blue)

2. Selector Elemen

Selector elemen digunakan untuk memberikan gaya berdasarkan nama tag. Misalnya, jika kita ingin mengubah ukuran font menjadi 16px untuk elemen paragraf, maka kodenya akan seperti ini:

p {
 font-size: 16px;
}

3. Selector ID

Selector ID digunakan untuk memberikan gaya berdasarkan id tertentu (#). Misalnya, di file index.html ada elemen dengan id: judul

<p id="judul">Surat Kabar Harian </p> Jika kita ingin mengubah warna teks menjadi merah dan ukuran font menjadi 16px pada elemen yang memiliki id "judul", maka kodenya seperti ini: #judul {
 color: red;
 font-size: 16px;
}

4. Selector Kelas

Selector kelas memilih elemen berdasarkan atribut kelas. Kelas diawali dengan titik (.):

Misal pada file index.html terdapat paragraf dengan class= judul

<p class="judul">Surat Kabar Harian </p> Jika kita ingin mengubah warna teks menjadi merah dan ukuran font menjadi 16px pada elemen yang memiliki kelas "judul", maka kodenya seperti ini: .judul {
 color: red;
 font-size: 16px;
}
