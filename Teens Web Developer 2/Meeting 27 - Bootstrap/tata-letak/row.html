Penggunaan Kelas row dalam Bootstrap

Kelas row digunakan untuk membuat baris dalam grid Bootstrap. Setiap baris dapat memuat kolom yang diatur dalam ukuran dan lebar yang sesuai. Kelas ini membantu untuk menjaga struktur layout dan membuat elemen-elemen di dalamnya tersusun rapi.

Setiap row secara otomatis mengatur margin negatif di sisi kiri dan kanan, untuk mengimbangi padding pada kolom. Ini memastikan bahwa kolom akan selalu berada di tepi kontainer.
